### 01背包问题

背包体积 v
N件物品，i件体积v[i]，价值w[i]
求可以放入的最大价值。

f[i][j]： 第i个物品，在总体积为j时，背包能放入的最大价值

不能够放入：
    f[i][j] = f[i-1][j]
能够放入：
    放入i：f[i][j] = f[i - 1][j - v[i]] + w[i]
    不放入i：f[i][j] = f[i-1][j]

f[i][j] = max(f[i-1][j-v[i]], f[i-1][j])
